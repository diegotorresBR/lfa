Sistemas de Estados Finitos

Um Sistema de Estados Finitos é um modelo matemático de estradas e saídas discretas. Pode assumir um número finito e pré-definido de estados. Cada estado resume somente as informações dos estados necessárias para determinar as ações para a próxima entrada. Um forte motivacional para o estudo de Sistemas de Estados Finitos é o fato de poderem ser associados a diversos tipos de sistemas naturais e construídos.

Um exemplo clássico e de simples entendimento é o elevador. Trata-se de um sistema que memoriza as requisições anteriores. Cada “estado” sumariza as informações “andar corrente” e “direção de movimento”. As entradas para o sistema são requisições pendentes.
