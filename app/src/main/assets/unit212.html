Expressão regular

Uma expressão regular é uma palavra no alfabeto Σe, construída recursivamente pela aplicação sucessiva das seguintes regras:

• ∅ ∈ ER denota a linguagem vazia: ∅

• ε é ER denota a linguagem { ε }

• x é ER (para qualquer x ∈ Σ) denota a linguagem { x }

Passo de Indução: se r e s são ER e denotam as ling. R e S, então:

União. (r + s) é ER denota a linguagem R ∪ S

Concatenação. (rs) é ER denota a linguagem R S = { uv  u ∈ R e v ∈ S }

Concatenação Sucessiva. (r*) é ER denota a linguagem R*
